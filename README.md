# Cliking Tools

## Installation:
**add in file composer.json**
```javascript
"require": {
	"megaads/cliking-lara42": "^1.0"
}
```
## Usage:
**Register Provider**
```
#/Config/app.php
'providers' => [
    'Megaads\ClikingLara42\ClikingLara42ServiceProvider'
];
```

**Access report**

`http://domain.name/logger`
